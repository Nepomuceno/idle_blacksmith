## [1.0.0] - 2026-01-03

### Added

#### Core Gameplay
- **Forge System** - Tap to forge weapons with satisfying feedback
- **8 Weapon Tiers** - Common, Uncommon, Rare, Epic, Legendary, Mythic, Divine, Eternal
- **Gold Economy** - Earn gold from forging and passive income
- **Auto-Forge** - Automatic forging when unlocked

#### Progression Systems
- **Upgrades** - 10+ upgrades to boost click power, passive income, and auto-forge
- **Achievements** - Unlock rewards for reaching milestones
- **Ascension** - Prestige system with Ancient Souls
- **Soul Shop** - Permanent upgrades that persist across ascensions
- **Weapon Upgrades** - Per-weapon type multipliers

#### User Interface
- **Responsive Layout** - Adapts to mobile (portrait) and desktop (landscape)
- **Tab Navigation** - Forge, Upgrades, Achievements, Shop tabs
- **Animated Splash Screen** - With lore snippets and loading animation
- **Visual Feedback** - Animations for forging, upgrades, and achievements

#### Technical
- **Save System** - Auto-save every 30 seconds with JSON format
- **Offline Progress** - Earn passive income while away
- **Cross-Platform** - Windows, macOS, Linux, iOS, Android support
- **Export Presets** - Ready-to-build configurations for all platforms

#### Content
- **World Lore** - The realm of Aethermoor and the Eternal Forge
- **Player Story** - Last apprentice of Master Thornwick
- **Tier Lore** - Unique descriptions and flavor text for each tier
- **Loading Tips** - Helpful hints during loading

#### Documentation
- **README** - Project overview and setup instructions
- **Build Guide** - Detailed export instructions for all platforms
- **App Store Guide** - Complete iOS submission walkthrough (Portuguese)
- **Privacy Policy** - For app store submissions
- **Contributing Guide** - For open source contributors

### Technical Details
- Built with Godot 4.5
- GDScript codebase
- GUT testing framework
- OpenGL ES 3.0 / Compatibility renderer
